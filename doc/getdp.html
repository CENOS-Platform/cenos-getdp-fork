Description

GetDP is a scientific computation software for the numerical solution of integro-differential equations, using finite element and integral type methods. The main feature of GetDP is the closeness between the input data defining discrete problems (written by the user in ASCII data files) and the symbolic mathematical expressions of these problems.

Mailing lists

  • getdp is the public mailing list for GetDP users. You should send all questions, bug reports, requests or pleas for changes related to GetDP to this list. The list is archived here
  • getdp-announce is a moderated (i.e. "read-only") list for announcements about significant GetDP events. You should subscribe to this list to get information about software releases, important bug fixes and other GetDP-specific news. The list is archived here.

Download
